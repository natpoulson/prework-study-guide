HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Semantic HTML refers to elements that describe the context of their contents. This is useful for assistive technologies such as screen readers, as it helps organise the order in which content is presented or narrated.
  • The h1 to h6 tags are used to mark headings for content. As a rule-of-thumb, the h1 tag should only be used once.
  • All HTML elements can have attributes, which are included in the opening section of a tag. Examples of attributes include "class", "id", and "href".
  • The position of the opening and closing tags (on elements that use them) are used to indicate the hierarchy of elements. Elements placed in the space between another element's tags are considered to be children of that element. An example of such a relationship is the "body" tag, which contains the majority of a page's content.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • Using the auto value for margin will cause it to automatically take up available space on the side(s) it was specified for, provided a fixed height and width are defined.
  • padding indicates how much space we want around the content inside an element.
  • The representation of the components within an element is commonly referred to as the "box model".
  • From outermost to innermost, the box model comprises of: margins, borders, padding, and content.
  • The box-sizing property dictates how the sizes of elements are calculated in the browser, and they come in two different modes:
    • content-box (Default) - Calculates element size based on the content only, both padding and margins are calculated independently meaning the box itself can be much larger than its defined width and height.
    • border-box - Calculates element size up to the border of the box. The padding shares space with the content with its value representing a fixed or relative ratio of space between the content and the border, and the margin defines the space between the element and others.
  • The asterisk * is a universal selector and applies to all elements.
  • Multiple elements can be selected in one rule by delimiting each one with a comma. This can be used in conjunction with elements that have ID, Class, and other selectors.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add <file or folder>: marks a file for staging
  • git commit -m "<comment>": capture all staged files as a snapshot
  • git push <remote alias> <branch>: upload local commits to remote repository
  • git pull --all: download the latest version of all remote repositories and branches
